# White Label Templates

## Overview

BlockIDâ€™s White Label feature provides a way to customize the look and feel of the GUI for the specific pages as described below. It provides templates with the settings to give administrators the ability to add their CSS style code and brand it as per their requirement.

## <a name = "LoginT"></a>Login Template:
The Login Template screen allows administrators to rebrand the following four sub templates:
* OTP
* Broker
* LDAP
* OneKud

These templates will be available as per the list of authentication modules enabled for your community.

To view the Login template screen, perform the following steps:
* Login to **BlockID Admin Console**, navigate to ***Administration Console** > **White Label** > **Login Template***.

![WhiteLabelLoginTemplate](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/WhiteLabelLoginTemplate.jpg)

* In the **Login Template** screen:
    * **Template Name**: select the appropriate template from the list.
    * In the **Login Page** section:
        * In the **Login Template** code editor, enter the appropriate CSS style code for the selected sub template.
    * Click **Save**.

## <a name = "EmailT"></a>Email Template:
The Email Template screen allows administrators to customize the content formats for the following two sub templates:
* OTP
* Send OTP

To view the Email Template screen, navigate to ***Administration Console** > **White Label** > **Email Template***.

![WhiteLabelEmailTemplatewithTemplatetype](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/WhiteLabelEmailTemplatewithTemplatetype.jpg)

* In the Email Template screen:
    * **Template Name**: Select the appropriate template from the list.
    * In the **Email Template** section:
      * In the **Email Template** code editor, enter the appropriate CSS style code for the selected sub template.
    * Click **Save**.

## <a name = "ErrorT"></a>Error Page Template:
The Error Page Template screen allows administrators to customize the error message for the following sub templates:
* 400
* 401
* 403

To view the Error Page Template screen, navigate to ***Administration Console** > **White Label** > **Error Page Template***.

![WhiteLabelErrorPageTemplates](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/WhiteLabelErrorPageTemplates.jpg)

* In the Error Page Template screen:
    * Template Name: Select the appropriate template from the list.
    * In the Error Page Template section:
      * In the **Error Page Template** code editor, enter the appropriate CSS style code for the selected sub template.
    * Click **Save**.

## <a name = "AdminT"></a>Admin Template:
The Admin Template screen allows administrators to rebrand the logo and add custom CSS styles for the BlockID Admin Console of a particular community. 

To view the Admin Template screen, navigate to ***Administration Console** > **White Label** > **Admin Template***.

![WhiteLabelAdminTemplate](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/WhiteLabelAdminTemplate.jpg)
* In the **Admin Template** screen:
    * Upload Logo: Click **Choose File**, locate, and select the appropriate logo image (in `.png` format) file.
        * In the Custom CSS code editor, enter the appropriate CSS style code for the template.
    * Click **Save**.

## <a name = "EULAT"></a>End User License Agreement (EULA) Template:
The EULA Template screen offers you to restructure the logo and CSS for the EULA template.

To view the EULA Template screen, navigate to ***Administration Console** > **White Label** > **EULA Template***.

![WhiteLabelEULATemplates](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/WhiteLabelEULATemplates.jpg)

* In the **EULA Template** screen:
    * In the **End User License Agreement** code editor, enter the appropriate agreement for the template.
    * Click **Save**.
