# BlockID Developer API

The BlockID Developer API provides operations to interact with all BlockID applications and services. Designed to be secure and scalable, our language-agnostic API uses [ECDSA encryption](/docs/API/ecdsa) and implements modern REST principles according to OpenAPI Specification standards.
