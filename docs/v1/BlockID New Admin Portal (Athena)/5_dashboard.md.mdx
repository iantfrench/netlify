# Dashboard

The Dashboard is a landing page that provides a quick view of multiple sections available within the Athena Admin Console. To check the Dashboard screen, log in to the Athena Admin Console application as an Administrator, the Dashboard screen is displayed.

In the Dashboard screen, you can:

1.  Explore knowledge center
2.  Add session attributes
3.  View email templates
4.  Onboard SAMLv2 applications such as Salesforce, Okta, Auth0, and many more for passwordless authentication
