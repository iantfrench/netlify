# Directory
The administrator can add and manage connections to your user directory sources. This will achieved by adding the directory server details and Service account configurations to establish a connection to the required user directory.

This guide describes the following procedures:

1. [Add a new user directory source](#Adddir)
2. [View, edit and delete a user directory sources](#managedir)

## <a name = "Adddir"></a>Add a new user directory source
To add a new user directory soruce, perform the following steps:

1. Login to the Athena Admin portal, from the left pane, click on the **Directory** icon. The **Directory Integrations** screen is displayed.
![DirectoryIntegrationsscreen.png](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/DirectoryIntegrationsscreen%282%29.png)

2. In the **Directory Integrations** screen, click **Add new**. The **Add Directory** screen is displayed.
![AddDirectoryscreen.png](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/AddDirectoryscreen.png)

3. In the **Add Directory** screen, enter the following details:
    * In the **Connect to a directory** section:
        * **Directory type**: Select the directory type. For example, `LDAP` or `AD`.
        * **Directory name**: Enter the appropriate name for your directory.
    * In the **Connection Information** section:
        * **Server protocol type**: Select the **LDAP** or **LDAPS (LDAP Secure)** protocol type of your server where the user directory is located.
        * **Server URL**: Enter the appropriate directory server URL. 
        * **Server port**: Enter the appropriate directory server port number.
        * **Base Domain** Enter the appropriate directory domain details. For example, `dc=yourcompany,dc=local`.
    * In the **Service Account Information** section:
        * **Service account username**: Enter the appropriate service account name from which the user directory information will be accessed. 
        * **Service account password**: Enter the appropriate password for the selected service account.
        * **Service account read only**: Select the option to enable this field. This will restrict editing of the the service account information.
        * **Search control**: Select the level option to search for the users.
        * **Authentication method**: Select the appropriate authentication method.
        * **Search attribute**: Enter the search attribute details.
        * **Filter**: Enter the filter options.
    * Click **Connect**. The newly added user directory source is displayed in the list of directory sources on the **Directory Integrations** screen. 

## <a name = "managedir"></a>View, edit and delete a user directory sources

To add a new user directory soruce, perform the following steps:

1. Login to the Athena Admin portal, from the left pane, click on the **Directory** icon. The **Directory Integrations** screen is displayed.
![DirectoryIntegrationsscreen.png](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/DirectoryIntegrationsscreen%282%29.png)

2. In the **Directory Integrations** screen:
    * Click on the **Edit** option from the **Actions** column to update the configurations for the desired directory source.
    * Click on the **Delete** option to remove the required directory source.
