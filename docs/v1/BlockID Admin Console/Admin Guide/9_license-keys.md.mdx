# License Keys
In the License Keys screen, you can generate a new license key, and select the required modules to be used for the production and development environments. This license key will be utilized by the BlockID mobile application.

You can view the existing license key and the list of enabled modules added to it. There are options to add or remove modules and update the expiry date of your existing license key.

**List of Topics:**
1. [Generate a new License Key](#GenerateLicenseKey)
2. [View, Edit and Delete a particular License Key](#VEDLicenseKey)
3. [View BlockID Verify SDK Documentation](#BlockIDSDK)


To view the License Keys screen, perform the following steps:
1. Login to **BlockID Admin Console**, navigate to ***Administration Console** > **License Keys***. 

![LicenseKeysscreen.jpg](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/LicenseKeysscreen%281%29.jpg)

In the **License Keys** screen, perform the following operations: 

## <a name = "GenerateLicenseKey"></a>Generate a new License Key:

1. Click **Generate Key**.
2. In the Create New License Key screen:
  ![GenerateLicenseKeysscreenwithgeneratedLicense.jpg](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/GenerateLicenseKeysscreenwithgeneratedLicense%281%29.jpg)


3. Select the checkbox appropriate modules. This will enable the selected modules for the customerâ€™s mobile application.
4. Click on the Calendar menu and select the appropriate Expiry Date for this license key. 
Click **Generate**. The new license key will be displayed in the **License Key** box.
    * Click **Save**. The newly created license key is displayed in the list of license keys.

![LicenseKeysscreenwithNewlygeneratedLicensekwy.jpg](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/LicenseKeysscreenwithNewlygeneratedLicensekwy%281%29.jpg)

:::(Info) (Note:)
When you generate a new license key, it will not be in use until you delete the existing key and add the new license key in your BlockID mobile application.
:::

## <a name = "VEDLicenseKey"></a>View, Edit, and Delete a particular License Key:
1. In the License keys list, click on the desired license key.  

![UpdateLicenseKeysscreen.jpg](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/UpdateLicenseKeysscreen%281%29.jpg)

2. In the **Update License Key** screen, perform the desired changes and click **Update**.
3. In the License Key screen, click on the **Delete** ![Deleteicon](https://cdn.document360.io/aeae7c88-d2f4-4a9c-bba9-c4f368260d78/Images/Documentation/Deleteicon.jpg) icon to remove the license key from the list, if required.
