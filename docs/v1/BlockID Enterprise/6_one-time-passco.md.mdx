## <a name = "OneTimePasscode"></a>One Time Passcode (OTP) Module
#### How to setup the One Time Passcode Module
The One Time Passcode is where you would configure Email, SMS, and Hardware Token (OneSpan) OTP 

Email OTP: (enter the following parameters)
Protocal: SMTP, IMAP)
Auth: True, False
Host (SMTP Gateway)
Port Number
From Email Address
Username
Password
Time Skew (in seconds)

SMS OTP: (enter the following parameters)
Account SID
Auth Token
Phone Number

Hardware Token Cfiguration 
Domain (OneSpan server)
