# BlockID User Interface Templates

1. Login Templates
2. Email Templates
3. Error Page Templates
4. Admin Templates
5. EULA Templates


* * *
:::tip [Login Templates](#LoginTemplate)
Within the Login Templates there are 5 Sub Templates

1. OTP
2. Broker
3. LDAP
4. OneKud
5. OTP

:::

:::tip [Email Template](#EmailTemplate)
Within the Email Templates there are 3 Sub Templates

1. Select Templates
2. OTP
3. Send OTP
:::


:::tip [Error Page Templates](#ErrorPageTemplates)
Within the Error Page Templates there are 3 Sub Templates

1. 400
2. 401
3. 403
:::
:::tip [Admin Templates](#AdminTemplates)
Within the Admin Templates there are 2 options

1. Upload Logo - You can insert your company logo to be displayed within the login BlockID loging page
2. Custom CSS
:::

:::tip [EULA Templates](#EULATemplates)
Within the EULA (End-user license agreement) Templates you can enter a custom EULA that applies to you and your users
:::
